# Frontend (Sprint 1)

## Реализовано

- Chat UI (React + Vite)
- Welcome screen
- Message list + input
- SSE streaming from backend
- Typing indicator
- Entry context detection (portal / telegram / external)

## Запуск

```bash
npm install
cp .env.example .env
npm run dev
```

`VITE_API_BASE_URL` by default: `http://127.0.0.1:8009`.
