## Что мы строим

Роль «Менеджер клиентского сервиса» для агента Эврика. Та же Эврика, что и продавец, но в режиме поддержки. Переключается по зданию на портале или по параметру `agent_role=support`.

**Общий код:** бэкенд и фронтенд общие с продавцом (`/eurika/backend/`, `/eurika/frontend/`).

**Главный документ:** `TZ.md`

## Идентичность агента

- **Имя:** Эврика
- **Роль:** Менеджер клиентского сервиса EdPalm
- **Тон с родителями:** спокойно, по делу, без официоза
- **Тон с учениками 1–8:** дружелюбно и понятно
- **Тон с учениками 9–11:** на равных, эмодзи уместны
- **Тон с менеджерами КС:** только алерты и задачи
- **Жёсткие правила:** не раскрывать данные других клиентов, отвечать только из базы знаний

## Сценарии

| Сценарий | Описание |
|---|---|
| А — Чек-лист | Клиент спрашивает по шагу процесса → агент объясняет |
| Б — Обращение | Входящий вопрос → RAG → ответ → NPS |
| В — Онбординг | Оплата → приветствие → проверка через 24ч |
