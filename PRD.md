---

## Что это

Эврика — ИИ-агент EdPalm, который работает в трёх ролях из одного кода. Выглядит как ChatGPT, отвечает за секунды, работает 24/7.

| Роль | Здание на портале | Что делает |
|---|---|---|
| **Продавец** | Магазин | Консультирует, подбирает продукт, обрабатывает возражения, закрывает сделки |
| **Поддержка** | Секретарь | Сопровождает по чек-листам, отвечает на вопросы, создаёт обращения |
| **Учитель** | Интерактивный город | Объясняет учебный материал, готовит к аттестациям, генерирует материалы |

Переключение ролей — по зданию на портале или по параметру `agent_role`.

---

## Где работает

- **Портал EdPalm** — здание на интерактивной карте → чат с агентом (JWT-авторизация)
- **Telegram Mini App** — бот `miniapp_edpalm_bot`, нативный опыт
- **Внешняя ссылка** — для новых клиентов из рекламы, WhatsApp, рассылок

Все три точки входа — один и тот же агент, один код, разные оболочки.

## Роль: Продавец

**Сценарии:**
1. **Новый клиент** — квалификация → подбор продукта → возражения → оплата → follow-up
2. **Пролонгация** — триггер из CRM → NPS → предложение → оплата
3. **Кросс-сейл** — доп.продукты в ходе диалога

**Инструменты:** RAG-поиск, amoCRM (контакты, сделки), DMS (платёжные ссылки), эскалация, follow-up цепочка (24ч/48ч/7д).

**Эскалация:** негатив, просьба о человеке, нет ответа в KB, скидка не из KB, 3 follow-up без ответа.

**Дашборд:** конверсии, GMV, эскалации, топ незакрытых вопросов.

Детали: `seller_staff/TZ.md`

---

## Роль: Поддержка

**Сценарии:**
- А — Чек-лист: клиент спрашивает по шагу процесса → агент объясняет
- Б — Обращение: вопрос → RAG → ответ → NPS
- В — Онбординг: оплата → приветствие → проверка через 24ч
- Г — Уведомления: оплата, занятия, ДЗ, документы, зачисление
- Д — Алерты: клиент молчит 48ч, падение успеваемости → менеджеру КС
- Е — ГИА: сроки, чеклисты, FAQ для 9–11 классов

**Инструменты:** RAG-поиск по FAQ, создание обращений в amoCRM (Service pipeline), DMS (профиль клиента), Telegram-уведомления, NPS, автотегирование.

**Дашборд:** обращения, топ тем, NPS, уведомления, алерты, еженедельный отчёт.

Детали: `support_staff/TZ.md`

---

## Роль: Учитель

**Сценарии:**
- А — Вопрос по теме: ответ на основе учебников «Просвещения»
- Б — Подготовка к аттестации: разбор тем + напоминание за день
- В — Напоминания: за 10 мин до урока, через 2 дня без активности
- Г — Генерация материалов: квиз, карточки, визуальная карта, аудиопересказ

**Источники:** учебники «Просвещения» (RAG), профиль ученика из ЛК, расписание (после автоматизации).

**Ограничения:** задания, успеваемость и посещаемость недоступны (ограничения LMS).

Детали: `teacher_staff/TZ.md`
